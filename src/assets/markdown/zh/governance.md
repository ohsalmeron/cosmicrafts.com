## 简介

Cosmicrafts DAO将社区置于中心位置，赋予利益相关者对游戏系列如何发展的真正发言权。基于成熟技术构建，DAO通过公平、透明和社区驱动的决策来确保Cosmicrafts坚持其愿景。

::: info 阅读指南
本文档概述了Cosmicrafts DAO的治理框架，重点关注决策流程、提案系统和社区参与。它是[代币经济](/tokenomics)文档的补充，后者涵盖了经济方面。

- **主要焦点**：治理流程、投票和社区决策
- **配套文档**：[代币经济](/tokenomics)了解代币经济和效用
- **交叉引用**：查看提示框，链接到相关的代币经济部分
:::

::: info 动态投票权
投票权的实际分配将根据利益相关者的以下决策而波动：
- 在神经元中质押的代币数量
- 所选择的溶解延迟长度
- 累积的神经元年龄
这些因素可以随着时间的推移显著改变不同利益相关者群体的相对影响力。
:::

| 利益相关者群体 | 投票份额 | 基础代币 | 目的 |
|-------------------|--------------|-------------|----------|
| **SNS参与者** | 50% | 1.2亿 | - 最大投票集团<br>- 社区驱动的治理<br>- 通过参与提高影响力的潜力 |
| **开发团队** | 33.3% | 8000万 | - 战略决策<br>- 8年溶解延迟，1年归属期<br>- 影响力逐渐减少 |
| **创世/种子** | 16.7% | 4000万 | - 早期支持者代表<br>- 错开的溶解延迟（0-7年）<br>- 平衡的初始影响力 |

### 权力乘数

| 因素 | 最大奖励 | 达成时间 |
|--------|---------------|-----------------|
| **溶解延迟** | +100% | 8年 |
| **神经元年龄** | +100% | 1年 |
| **最低溶解延迟** | 不适用 | 1个月 |
| **组合上限** | 基础权力的3倍 | 不适用 |

## SNS整合

Cosmicrafts DAO利用Internet Computer的[服务神经系统（SNS）](https://internetcomputer.org/docs/building-apps/governing-apps/overview)作为其治理基础设施。这为去中心化决策提供了经过实战检验的安全框架。

### 主要特点

- **基于神经元的投票**：质押SPIRAL代币创建神经元并参与治理
- **溶解延迟奖励**：更长的锁定期增加投票权
- **年龄奖励**：神经元随着时间增加投票权
- **提案系统**：所有治理行动的标准SNS提案框架

::: info SNS配置
有关我们SNS设置的详细技术参数，包括最低质押、投票期和奖励率，请参阅我们文档中的SNS初始化参数。
:::

### 治理参数

| 参数 | 值 | 目的 |
|-----------|--------|---------|
| **拒绝费用** | 1000 SPIRAL | 防止垃圾提案 |
| **初始投票期** | 7天 | 标准审议时间 |
| **最长截止日期延长** | 1天 | 允许延迟参与 |
| **最低神经元创建质押** | 1000 SPIRAL | 基本参与门槛 |

## 神经元质押机制

### 基本要求

| 参数 | 值 | 描述 |
|-----------|--------|-------------|
| **最低质押** | 1,000 SPIRAL | 创建神经元所需的基本数量 |
| **最短锁定期** | 30天 | 最短允许的溶解延迟 |
| **最长锁定期** | 8年 | 最长可能的溶解延迟 |
| **交易费** | 0.01 SPIRAL | 网络运营成本 |

### 成熟度计划

| 锁定期 | 奖励乘数 | 有效权力 |
|-------------|------------------|-----------------|
| 30天 | 1.0x | 基础权力 |
| 6个月 | 1.25x | +25% |
| 1年 | 1.5x | +50% |
| 2年 | 1.75x | +75% |
| 4年 | 1.85x | +85% |
| 8年 | 2.0x | +100% |

## 决策框架

### 治理领域

1. **金库管理**
   - 营销活动
   - 开发资金
   - 战略合作伙伴关系

2. **经济政策**
   - 代币经济调整
   - 质押利率
   - 费用结构

3. **开发路线图**
   - 功能优先级
   - 游戏扩展
   - 技术改进

### 提案系统

| 阶段 | 持续时间 | 要求 |
|-------|----------|--------------|
| **提交** | 不适用 | 1,000 SPIRAL质押 |
| **审查** | 24小时 | 社区反馈 |
| **投票** | 7天 | 需要激活的神经元 |
| **执行** | 可变 | 如果批准则自动执行 |
