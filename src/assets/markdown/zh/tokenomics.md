# 代币经济
![代币经济](spiralbanner.webp)
## 概述

SPIRAL代币是Cosmicrafts DAO的经济基础，旨在：
- 推动治理和奖励参与
- 驱动可持续的经济增长
- 通过结构化激励机制调整利益相关者利益
- 实现跨游戏价值转移和效用

总供应量：**10亿 SPIRAL代币**

分配给Cosmicrafts基金会
- **目的**：确保长期承诺和价值创造
- **结构**：8年溶解延迟
- **治理**：锁定代币拥有完全投票权

#### Anmi Capital (1000万)
区块链行业专业知识和网络
- 8个神经元，每个125万
- 季度归属计划

#### Genesis (1400万)
早期支持者神经元篮子
- 8个神经元，每个175万
- 年度归属计划

### 公共分配 (1.2亿)
- **结构**：每3个月进行一次共8次的归属事件
- **释放**：每次事件12.5%（1500万SPIRAL）
- **目的**：确保稳定的代币分配
- **参与限制**：
  - 最小：100 ICP
  - 最大：每参与者100,000 ICP
  - 总最大：1,000,000 ICP

## 效用框架

### 治理功能

| 功能 | 描述 | 要求 |
|---------|-------------|--------------|
| **提案创建** | 提交新提案供社区投票 | 最低1,000 SPIRAL质押 |
| **投票权** | 参与DAO决策制定 | 需要激活的神经元 |
| **协议参数** | 对系统范围的参数变更投票 | 积极参与治理 |

### 经济效用

| 功能 | 价值 | 目的 |
|----------|--------|----------|
| **交易费** | 0.01 SPIRAL | 网络维护和通缩 |
| **跨游戏交易** | 可变 | 实现游戏间价值转移 |
| **NFT操作** | 基于市场 | 促进数字资产交易 |
| **高级功能** | 基于层级 | 访问高级平台功能 |

### 社区福利

| 福利 | 类型 | 详情 |
|---------|------|----------|
| **质押奖励** | 经济 | 基于锁定期的动态年收益率 |
| **锦标赛访问** | 游戏 | 优先参加竞技活动 |
| **抢先体验** | 产品 | 新功能和游戏预览 |
| **特别活动** | 社交 | 专属社区聚会和活动 |

## 奖励
奖励结构旨在：
- 鼓励长期参与
- 协调利益相关者利益
- 维持可持续的代币经济
- 支持协议治理
- 实现社区驱动的价值分配
