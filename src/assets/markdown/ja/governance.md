## はじめに

Cosmicrafts DAOはコミュニティを中心に置き、ステークホルダーにフランチャイズの成長方法について実質的な発言権を与えます。実証済みの技術に基づいて構築されたDAOは、公平性、透明性、およびコミュニティ主導の意思決定を通じて、Cosmicraftsがそのビジョンに忠実であることを確保します。

::: info 読書ガイド
この文書はCosmicrafts DAOのガバナンスフレームワークを概説し、意思決定プロセス、提案システム、およびコミュニティ参加に焦点を当てています。経済的側面をカバーする[トークノミクス](/tokenomics)文書を補完するものです。

- **主な焦点**: ガバナンスプロセス、投票、コミュニティ意思決定
- **関連文書**: トークン経済とユーティリティについては[トークノミクス](/tokenomics)を参照
- **相互参照**: 関連するトークノミクスセクションへのリンクを含むヒントボックスを探してください
:::

::: info 動的投票力
投票力の実際の分配は、ステークホルダーの以下の決定に基づいて変動します：
- ニューロンにステークされたトークンの量
- 選択された溶解遅延の長さ
- 蓄積されたニューロンの年齢
これらの要因は、時間の経過とともに異なるステークホルダーグループの相対的な影響力を大きく変えることがあります。
:::

| ステークホルダーグループ | 投票シェア | 基本トークン | 目的 |
|-------------------|--------------|-------------|----------|
| **SNS参加者** | 50% | 1.2億 | - 最大の投票ブロック<br>- コミュニティ主導のガバナンス<br>- 参加を通じた影響力増大の可能性 |
| **開発チーム** | 33.3% | 8000万 | - 戦略的意思決定<br>- 1年間のベスティングを伴う8年間の溶解遅延<br>- 影響力の段階的な低減 |
| **ジェネシス/シード** | 16.7% | 4000万 | - 初期支援者の代表<br>- 段階的な溶解遅延（0-7年）<br>- バランスの取れた初期影響力 |



### パワー乗数

| 要素 | 最大ボーナス | 達成時間 |
|--------|---------------|-----------------|
| **溶解遅延** | +100% | 8年 |
| **ニューロン年齢** | +100% | 1年 |
| **最小溶解遅延** | 該当なし | 1ヶ月 |
| **組み合わせ上限** | 基本パワーの3倍 | 該当なし |

## SNS統合

Cosmicrafts DAOはインターネットコンピュータの[サービスナーバスシステム（SNS）](https://internetcomputer.org/docs/building-apps/governing-apps/overview)をガバナンスインフラストラクチャとして活用しています。これにより、分散型意思決定のための戦闘テスト済みの安全なフレームワークが提供されます。

### 主な特徴

- **ニューロンベースの投票**: SPIRALトークンをステークしてニューロンを作成し、ガバナンスに参加
- **溶解遅延ボーナス**: 長期のロックアップ期間により投票力が増加
- **年齢ボーナス**: ニューロンは時間の経過とともに投票力を獲得
- **提案システム**: すべてのガバナンスアクションに標準SNS提案フレームワークを使用

::: info SNS設定
最小ステーク、投票期間、報酬率を含むSNSセットアップの詳細な技術パラメータについては、ドキュメントのSNS初期化パラメータを参照してください。
:::

### ガバナンスパラメータ

| パラメータ | 値 | 目的 |
|-----------|--------|---------|
| **拒否料金** | 1000 SPIRAL | スパム提案の防止 |
| **初期投票期間** | 7日 | 標準的な審議時間 |
| **最大期限延長** | 1日 | 遅延参加を許可 |
| **最小ニューロン作成ステーク** | 1000 SPIRAL | 基本参加閾値 |


## ニューロンステーキングメカニクス

### 基本要件

| パラメータ | 値 | 説明 |
|-----------|--------|-------------|
| **最小ステーク** | 1,000 SPIRAL | ニューロンを作成するために必要な基本金額 |
| **最小ロック期間** | 30日 | 最短許容溶解遅延 |
| **最大ロック期間** | 8年 | 最長可能溶解遅延 |
| **取引手数料** | 0.01 SPIRAL | ネットワーク運用コスト |

### 成熟度スケジュール

| ロック期間 | ボーナス乗数 | 有効パワー |
|-------------|------------------|-----------------|
| 30日 | 1.0x | 基本パワー |
| 6ヶ月 | 1.25x | +25% |
| 1年 | 1.5x | +50% |
| 2年 | 1.75x | +75% |
| 4年 | 1.85x | +85% |
| 8年 | 2.0x | +100% |


## 意思決定フレームワーク

### ガバナンス領域

1. **資金管理**
   - マーケティングキャンペーン
   - 開発資金
   - 戦略的パートナーシップ

2. **経済政策**
   - トークノミクス調整
   - ステーキングレート
   - 手数料構造

3. **開発ロードマップ**
   - 機能優先順位付け
   - ゲーム拡張
   - 技術改善

### 提案システム

| 段階 | 期間 | 要件 |
|-------|----------|--------------|
| **提出** | 該当なし | 1,000 SPIRALステーク |
| **レビュー** | 24時間 | コミュニティフィードバック |
| **投票** | 7日 | アクティブなニューロンが必要 |
| **実行** | 変動 | 承認されれば自動化 |
